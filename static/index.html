Tumbo

Highly flexible Application Runtime Platform

Focus on principal tasks - gain success and reduce effort

Tumbo is a Server Platform for simplifying common development and deployment tasks. It conduce to go live quickly with an application - with less deployment- and configuration requirements.

Tumbo is Open source and can be extended by anybody.

The Tumbo Stack is based on Linux, Django and Docker

Features What you do with Tumbo

Run Python

Write Python code in the browser or provide a GIT Repo and execute the code from anywhere.

Asynchronous and Scheduled

Execute Long-running tasks asynchronously and get the results later. Execute task periodically.

Static Content

Load static files with dynamic parts from an installed Python module or from the Git repository.

CI/CD

Deploy projects to another Tumbo Platform. Integrate Tumbo in other systems and workflows.

Develop with Tumbo

Install Tumbo as Runtime Environment on your notebook for development.

Client JS Framework

Use our javascript library for Angular integration.

Datastore

Simple API for persisting Data in JSON. Access the data in the Console, Staticfiles or Controllers.

Run Custom Applications

Start any service or application on the worker and use the networking feature.

Versions Compare

Development Runserver

For Core and Tumbo App Development on your Linux Machine:

$ git clone https://github.com/sahlinet/tumbo-server.git && cd tumbo-server
$ cli/tumbo.py server dev run --autostart

On Premise Private

Running the full stack on your Docker Host

$ git clone https://github.com/sahlinet/tumbo-server.git && cd tumbo-server
$ cli/tumbo.py server docker run

For running Tumbo on a Kubernetes Cluster see the Run on Kubernetes Guide.
